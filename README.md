### Run the app (development)

first clone the repo
```
git clone (url_project)
```

install the dependencies
```
pnpm install
```

install nest cli
```
npm i -g @nestjs/cli
```

the docker command is used for run de mongo database
```docker
docker-compose up
```

run the up 
```
pnpm run start:dev
```
